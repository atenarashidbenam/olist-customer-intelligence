# Olist Customer Intelligence Platform

## Project Overview

The Olist Customer Intelligence Platform is an end-to-end data analytics and decision support project built on the Brazilian E-Commerce Public Dataset by Olist.

The project transforms raw marketplace data into a structured analytics platform that supports customer intelligence, business reporting, executive dashboards, and future machine learning applications.

Unlike a typical exploratory notebook, this project follows a production-oriented architecture consisting of data ingestion, ETL, relational database design, analytics engineering, performance optimization, business intelligence, and documentation.

---

## Objectives

The primary objectives of this project are to:

- Build a structured SQL Server data platform from raw CSV files.
- Apply consistent business rules across all analytical outputs.
- Analyze customer behavior using RFM methodology.
- Measure product, seller, delivery, and sales performance.
- Provide executive-level KPIs for decision-making.
- Prepare clean analytical datasets for Power BI and Machine Learning.
- Demonstrate practical Data Engineering and Analytics Engineering skills.

---

## Business Scope

To ensure consistency across all business metrics, revenue-related analyses are calculated **only for delivered orders**.

This rule is consistently applied throughout the project, including:

- Executive Dashboard
- Monthly Sales
- Customer RFM
- Product Performance
- Seller Performance

Orders with other statuses (such as canceled or unavailable) are excluded from finalized revenue calculations.
